## Odoo Development Environment
This repository provides a list of tools, configurations, modules and guidelines to set up a productive Odoo development environment for learning and teaching purposes.

### Prerequisites
- Basic knowledge of Python and Odoo framework
- Familiarity with Git and version control
- Access to a development machine (Windows, macOS, or Linux)
- Odoo version 18.0

### Tools and Technologies
- **Odoo**: The main framework for development
- **PostgreSQL**: The database management system used by Odoo
- **Python**: The programming language used for Odoo development
- **Git**: Version control system for managing code changes
- **Visual Studio Code**: Recommended code editor with Odoo extensions
- **Docker**: For containerized development environments

### Best Practices
- Use virtual environments for Python dependencies
- Regularly commit changes to Git
- Follow Odoo coding guidelines
